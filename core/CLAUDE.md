# Core

Главная программа для робота-футболиста.

## Робот

- Робот представляет из себя цилиндр радиусом 90мм и высотоый 178мм. Робот
  ездит на омниколесах - можно двигаться в любом направлении с любым поворотом.
- Компонентами нижнего уровня управляет arduino. У arduino отдельная программа.
- Камера смотрит снизу вверх на конусообразное зеркало, и таким образом робот
  видит во всех направлениях.

## Игра

Зеленое тканевое поле 243 x 182 см. Команды по 2 робота. Один робот -
нападающий, второй - вратарь.

## Прочее

- Когда добавляешь новое поле в конфиг, не трогай бэкапы конфигов (если они есть
в репозитории). Изменяй только config.example.json пока не сказано обратное
- Конфиги на роботах я обновлю руками. Не бойся, что программа от этого сломается
